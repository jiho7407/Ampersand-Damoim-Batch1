# 연구실 컨택 인사이더 상담 플랫폼

> 이공계 대학원 지원자와 재학 대학원생을 1:1 전화로 연결, 연구실 내부 정보의 비대칭을 해소합니다

## Tech Stack

- **Next.js 15** (App Router, TypeScript)
- **Tailwind CSS** + shadcn/ui
- **Supabase** (DB, Auth)
- **Vercel** (배포)

## Getting Started

```bash
npm install
cp .env.local.example .env.local
npm run dev
```
