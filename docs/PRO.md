# PRD — Peer Intel

> 버전: v0.1 (Pretotype)
> 최종 수정: 2025-04

---

## 1. 배경 및 목적

이공계 학부생이 대학원 연구실을 컨택하는 과정에서 신뢰할 수 있는 **내부 정보를 얻을 수단이 존재하지 않는다.**

- 김박사넷: 익명 텍스트, 신뢰도 낮음, 최근 검색 기능 붕괴
- 하이그래드넷: 공고 게시판 수준, 연구실 내부 정보 없음
- 고가 컨설팅: 50~200만 원, 접근 불가
- 에브리타임: 대학원생 접근 차단 구조

**본 서비스의 목적:** 소속 인증 대학원생과 학부생을 1:1 전화로 연결, 연구실 내부 정보 비대칭 해소.

---

## 2. 목표 사용자

### Primary: 학부생 (수요자)

- 이공계 학부 3~4학년
- 대학원 진학 준비 중
- 특히 **타대 지원자**, **컨택 경험이 없는 초보 지원자**
- 정보 접근이 가장 어려운 계층

### Secondary: 대학원생 (공급자)

- 이공계 석/박사 재학생
- 학교 이메일로 소속 인증
- 부수입 목적, 후배 도움 동기

---

## 3. 핵심 기능 (Pretotype 범위)

### 3.1 퍼널 페이지 (5단계)

| 단계 | 페이지        | 핵심 요소                |
| ---- | ------------- | ------------------------ |
| 1    | 랜딩          | 문제 공감, CTA           |
| 2    | 연구실 탐색   | 필터, 연구실 카드        |
| 3    | 대학원생 카드 | A/B/C/D 유형             |
| 4    | 상담 신청     | 가격 표시, 결제 버튼     |
| 5    | 대기 리스트   | 이메일 수집, "베타 안내" |

### 3.2 분석 트래킹

- 각 퍼널 단계별 이벤트 로깅
- 세션 ID 기반 익명 추적
- A/B 테스트 결과 집계

### 3.3 대기 리스트 수집

- 이름, 이메일, 전화번호(선택)
- 관심 분야, 클릭한 카드 유형 저장

## 5. 측정 지표

| 지표                 | 목표    | 의미                |
| -------------------- | ------- | ------------------- |
| 랜딩 CTA 클릭률      | 30%     | 문제 공감 확인      |
